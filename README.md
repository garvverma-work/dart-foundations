# dart-foundations
Dart from scratch — syntax, OOP, async, collections &amp; DSA. Foundation for Flutter development.

## Topics Covered

- **Basics** — variables, data types, null safety, type casting
- **Control Flow** — if/else, switch, loops, ternary
- **Functions** — named/optional params, arrow functions, higher-order functions
- **OOP** — classes, constructors, inheritance, abstract classes, mixins
- **Async** — Future, async/await, Streams
- **Collections** — List/Map/Set, functional methods (map, where, reduce)
